Using two data.frames or matrixs to create sample sets

Description

Using two data.frames or matrixs to create sample sets

Usage

Sample(x, y, count)

Arguments

x

A data.frame or matrix of expression profile

y

A data.frame or matrix of expression profile

count

A numeric variable that controls how many sample sets are produced

Value

20*count sample sets

Examples

Sample(GSE85589,GSE59856,100)#GSE85589 is a data.frame of expression profile,GSE59856 is also a data.frame of expression profile
